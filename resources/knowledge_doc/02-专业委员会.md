# 专业委员会

## 概述

专业委员会是学会下属的专业性学术组织，按研究方向划分。每个会员可选择加入一个专业委员会，并可关注多个专业委员会。

## 委员会角色

| 角色 | 职责 |
|------|------|
| 主任委员 | 负责委员会的全面工作，组织和领导学术活动 |
| 副主任委员 | 协助主任委员工作，分管特定领域 |
| 委员 | 参与委员会的学术活动和事务讨论 |

一个会员只能在一个委员会中担任角色，但可以同时关注多个委员会。

## 加入方式

- 入会申请时选择希望加入的委员会
- 已入会会员可申请变更委员会
- 关注委员会不限数量
