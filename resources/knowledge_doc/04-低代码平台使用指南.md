# 低代码平台使用指南

## 概述

YAAI 低代码平台是管理后台的可视化页面搭建工具，管理员可以通过配置组件来构建学会官网的各个页面。

## 页面搭建流程

1. 创建页面：填写页面名称、编码、路径和类型
2. 创建版本：为页面创建编辑版本，可从模板创建或新建空白版本
3. 编辑页面：在编辑器中添加、移动、配置页面上的各个组件
4. 配置内容：设置每个组件的属性、样式、布局和数据来源
5. 保存并发布：保存版本后点击发布，页面在前台即时生效

## 常用操作

- 修改已发布页面：创建新版本 → 编辑 → 发布，原版本自动归档
- 使用模板：从页面模板快速创建页面，减少重复搭建工作
- 复用片段：将常用区域（如联系方式、页脚）保存为片段，在多个页面中引用
- 菜单配置：设置网站导航菜单，支持多级菜单和页面链接

## 可用组件

平台支持首页轮播图、富文本内容、通知公告列表、快捷入口、新闻区块、新闻列表、服务展示、活动展示、会议页面、友情链接等多种组件。具体以系统实际配置为准。
