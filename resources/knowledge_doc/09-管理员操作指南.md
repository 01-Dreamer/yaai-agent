# 管理员操作指南

## 角色与权限

学会管理平台有多个管理员角色，不同角色对应不同的管理权限（详见管理员角色说明）。管理员账号需手动在后台分配，新注册用户默认为普通会员。

## 日常操作

### 会员审核
1. 进入后台 → 会员审核
2. 查看待审核的个人会员和单位会员列表
3. 点击详情查看完整资料
4. 核实资料完整性和真实性
5. 选择"通过"或"拒绝"
6. 通过后会员可进行缴费

### 新闻发布
1. 进入后台 → 新闻管理
2. 点击"新建新闻"
3. 填写标题、正文、选择分类
4. 上传封面图（可选）
5. 设置置顶和时间
6. 点击发布

### 页面管理
1. 进入后台 → 页面管理
2. 查看所有页面及其当前版本
3. 创建、修改或删除页面
4. 通过页面编辑器编辑页面内容
5. 发布版本后前台即时生效

### 轮播图管理
1. 进入后台 → 轮播图管理
2. 创建新轮播图，选择分组（如首页轮播）
3. 上传图片，设置标题和跳转链接
4. 设置生效时间范围
5. 调整排序

### 菜单管理
1. 进入后台 → 菜单管理
2. 创建新菜单项，选择链接类型（站内页面或外部链接）
3. 设置父级菜单可创建多级菜单
4. 调整顺序

## 操作日志

所有管理员操作会自动记录。管理员可通过后台查看操作日志，用于审计追溯。

## 注意事项

- 删除操作不可逆，请谨慎执行
- 页面发布后立即生效，建议先在草稿版本中确认
- 敏感操作前系统会弹出确认提示
